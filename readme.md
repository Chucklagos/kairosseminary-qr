# Generador de Códigos QR

Este proyecto es una herramienta desarrollada para la **Red de Desarrollo Sostenible (RDS)**, con el propósito de facilitar la generación de códigos QR personalizados. Los usuarios pueden crear códigos QR a partir de texto o enlaces, personalizar su color y descargarlos en formato PNG. Todo esto en una interfaz amigable, diseñada con la identidad visual de la RDS.

---

## Funcionalidades

1. **Generación de Códigos QR:**
   - Permite generar códigos QR a partir de texto o enlaces ingresados por el usuario.
   
2. **Personalización de Color:**
   - Incluye un selector de color para personalizar el color principal del código QR. Por defecto, se ofrece negro (`#000000`), pero se puede seleccionar cualquier color.

3. **Descarga en Formato PNG:**
   - Los códigos QR generados se pueden descargar como archivos de imagen en formato PNG con un simple clic.

---

## Paleta de Colores Utilizada

La herramienta emplea una paleta de colores inspirada en el logo de la **RDS**:

- **Verde oscuro (`#1D4F36`):** Usado para el texto principal y los bordes de los campos.
- **Verde claro (`#8BA51E`):** Usado para los botones principales y resaltar elementos importantes.
- **Azul (`#5293A5`):** Usado para los botones al pasar el cursor.
- **Blanco (`#ffffff`):** Usado como fondo para el contenedor y el código QR.

---

## Cómo Usar

1. **Abrir la herramienta:**
   - Abre el archivo HTML en cualquier navegador moderno.

2. **Ingresar texto o enlace:**
   - En el campo `Texto o enlace`, escribe el texto o el enlace que deseas convertir en un código QR.

3. **Seleccionar color:**
   - Usa el selector de color para elegir el color principal del código QR.

4. **Generar código QR:**
   - Haz clic en el botón **Generar QR**. El código QR se generará automáticamente en la parte inferior.

5. **Descargar código QR:**
   - Una vez generado el QR, el botón **Descargar QR** se habilitará. Haz clic en él para guardar el código QR como una imagen PNG.

---

## Requisitos

### Navegador Web:
- Cualquier navegador moderno con soporte para HTML5, CSS3 y JavaScript.

### Librerías Utilizadas:
